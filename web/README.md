# Frontend: React + Vite App

This is the frontend application for the project, built with React and bundled using Vite. It runs inside a Docker container for a consistent development experience.

## Prerequisites

* Docker and Docker Compose installed on your machine.

## Getting Started

The frontend is designed to be run as part of the broader Docker Compose setup.

1. **Start the application:**
   From the root of the project (where the `docker-compose.yml` is located), run:
   ```bash
   docker compose up web -d
